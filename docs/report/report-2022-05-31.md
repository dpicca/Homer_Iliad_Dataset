## Introduction

Dans le contexte de notre travail final de Web sémantique nous avons travaillé autour de sujet de représentation de profile psychologique des personnages d'Iliade d'Homére. Le livre d'Iliade est loin d'une fiction typique. Il s'agit d'un mélange de poésie, fiction et des documentations historiques.

L'épopée se déroule pendant la guerre de Troie et détaille les événements survenus pendant quelques semaines de la dixième et dernière année de la guerre.(https://fr.wikipedia.org/wiki/Iliade)

Une caractéristique étonnamment intéressante de l'Iliade est le grand nombre de personnages représentés (environ 750 personnages dont environ 49 personnages principaux), à la fois des personnages humains et des divinités. L'Homère a attribués à ces personnages les caractéristiques détaillées et cela donne une vision approfondie de la personnalité « archétypale » dans les anciennes histoires.
### Modélisation de psycho-profile de personnage d'Iliade
Comme discuté plus haut, nous avons travaillé sur la base de l'ontologie qui comportait déjà d'autres ontologies alignés et les diverses classes.
On y trouvais surtout la classe du personnage (ma:Agent), les classes des émotions (Emotion avec les sous-classes Basic et Complex), les traits de personnalité NEO-PI classe Personality_Profile (connu sous le nom "Big 5" avec les sous classes : Agreeableness, Conscientiousness, Extraversion, Openess et Neuroticism) ainsi que les rôles narratives (Hero, AntiHero, Vilain).

L'ontologie de base, utilise le NEO-PI typologie de personnalité, le modèle de trait psychologique mondialement reconnu et utilisé par beaucoup de chercheurs. (1) Dans l'ontologie de base cette typologie permet ensuite de construire les rôles narratifs de personnages.

Nous avons souhaité travailler sur cette base et apporter un autre aspect pour décrire les traits de personnalité d'un personnage littéraire. C'est pourquoi nous avons choisi de modéliser le profil Myers-Briggs qui est très connu, mais peu utilisé dans les recherches scientifiques.

Le Myers Briggs Type Indicator (MBTI) est un outil d'évaluation psychologique déterminant le type psychologique d'un sujet parmi seize types différents, suivant une méthode proposée en 1962 par Isabel Briggs Myers et Katherine Cook Briggs. (2) Il a été crée sur la base de théorie de Carl Gustav Jung qui a décrit les quatre fonctions psychologiques de base
-   Extraversion vs. introversion.
-   Sensation vs. intuition.
-   Thinking vs. feeling.
-   Judging vs. perceiving.

MBTI combine ces fonctions et ainsi les seize types psychologiques de base se sont développés qui décrivent les traits psychologiques suivant :

D'où tire-t-on son énergie : Extraversion (E) ou Introversion (I)
Comment perçoit-on les choses : Sensation (S) ou INtuition (N)
Comment prend-on ses décisions : Pensée (T) ou Sentiment (F)
Comment aborde-t-on le monde : Jugement (J) ou Perception (P) *(4.)*

L'indicateur MBTI identifie alors 16 grands types de personnalité à partir des deux préférences possibles sur chacune des quatre dimensions précédentes.

Pour construire notre ontologie nous avons donc besoin de modéliser la liaison entre les cinq classes suivantes :

les phrases (Structure) - les émotions (Emotion) - NEO-PI (Personality-Profile) - MBTI (MyersBriggsPersonality) - les rôles narratives (Narrative Role).

#### 5.1 Rélation entre MBTI et NEO-PI

Pour pouvoir construire notre ontologie et modéliser les relations, nous avons commencé par étudier les liaisons entre MBTI, NEO-PI et les rôles narratifs. Très rapidement, nous nous sommes heurté au manque de littérature scientifique qui travaille avec la typologie Myers-Briggs. Nous nous sommes appuyé sur les vieux recherches qui nous donnent quelques pistes pour construire ce modèle, mais malheureusement par le manque de connaissances nous ne pouvons pas exprimer les relations plus subtilement.

Le principal problème dans l'utilisation de MBTI est que cette typologie ne prend pas en compte certains aspects primordiaux de la personnalité comme par exemple, MBTI ne connais pas "Neurotisicm". De ce fait, il est impossible d'exprimer les traits de personnalité "négative" (par exemple pour décrire un "Vilain") et ainsi créer une liaison directe entre les caractéristiques de MBTI et les Rôles Narratifs. Nous avons dû chercher les autres moyens comment relier la typologie MBTI et les NarrativeRole.

Suite à ce problème-là, nous avons pu découvrir un ancien recherche qui établit une certaine corrélation entre NEO-PI et MBTI (5). Cette recherche nous permet de créer une relation entre la classe *"PersonalityProfile"* et la classe "*MyersBriggsPersonality*" par Object Property *"isCorrelatedWith"*.

En résumé, cette recherche met en relation MBTI et NEO-PI de façon suivant :

| NEO-PI | correlation | MBTI |
|---|---|---|
| Agreeableness |  | thinking-feeling (T-F) |
| Conscientiousness |  | thinking-feeling (T-F)<br>judging-perceiving (J-P) |
| Extraversion |  | extraversion-introversion (E-I) |
| Neuroticism |  |  |
| Openness |  | extraversion-introversion (E-I) <br>thinking-feeling (T-F)<br>judging-perceiving (J-P)<br>sensing-intuitive(S-N)<br> |
Tab 1:  Relation NEO-Pi et MBTI
Grâce à cette liaison nous pouvons exprimer les rôles narratifs par les traits de personnalité Myers Briggs.

#### 5.2 Les rélations entre les Emotions et MBTI

Pour produire un modèle de traits de caractère de personnage d'Iliade, nous avons également besoin de construire une relation entre la classe Emotion et le profil MBTI.

Dans la littérature scientifique, il existe une liaison directe que nous pouvons utiliser dans notre ontologie. Cette recherche (6)se base sur les recherches de Robert Plutchik et montre le lien entre les émotions de bases et les fonctions cognitifs MBTI. Les relations sont exprimées de manière suivante :

| EMOTIONS |  | FONCTIONS COGNITIVES |
|---|---|---|
| Fear | <=> | Introverted Thinking |
| Anticipation | <=> | Introverted Intuition |
| Sadness | <=> | Introverted Feeling |
| Disgust | <=> | Introverted Sensing |
| Anger | <=> | Extroverted Thinking |
| Surprise | <=> | Extroverted Intuition |
| Joy | <=> | Extroverted Feeling |
| Trust | <=> | Extroverted Sensing |
*Tab 2. Liaison entre les émotions et les fonctions cognitives* 

Nous avons exprimé cette relation directement comme un sous classe qui reunit ces deux aspects de personalité par un Objet property "hasEmotionalCognitiveFunction"*.

## Conclusion: 
Il est évidemment difficile et complexe d'exprimer les relations entre le texte et les émotions et plusieurs chercheurs essaient d'apporter les réponses depuis des décennies à ce sujet. Pour construire une ontologie, le "knowledge" est primordial. Nous nous rendons compte, que nos connaissances dans ce domaine sont très limitées. Nous vous proposons donc qu'une embauche de nos recherches à ce sujet.

Pour aller plus loin :

Notre travail autour de l'ontologie de profile psychologique des personnages de Iliade c'est concentré autour des relations autour de typologie MBTI. Comme cette ontologie a pour le but d'exprimer au mieux les traits de personnalité d'Iliade, nous voyons d'autres possibilités comment enrichir cette ontologie et aller plus loin.

Il est connu, que l'ancienne littérature grecque est rempli de mythologies, des déités et des archétypes. Nous pensons qu'il serait intéressant de traiter encore cette dimension. Carl Gustav Jung a fait le travail important dans le domaine mythologie et les archétypes domaine et peut être facilement rajouté. Egalement avec nous pouvons combiner les traits de caractère avec le tempérament et exprimer encore dans ontologie le tempérament (cholérique, sanguinique, mélancolique et flegmatique) et mettre en lien les émotions avec le tempérament.
